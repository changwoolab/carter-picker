# carter-picker

## Carter Picker란?

`랜덤 추첨 N명` 을 하고 싶을 때 사용할 수 있는 유틸 프로그램입니다.

## 준비물

아래와 같은 형식의 엑셀파일

- 이 때, 엑셀 파일명에 띄어쓰기가 들어가면 안 됩니다.

  - `추첨.xlsx` OK
  - `추첨 추첨.xlsx` BAD

결과 csv 파일에는 첫번째 column 값들만 들어갑니다.

## 설치

1. Node.js 설치: https://nodejs.org/en/download

# 사용법

1. 엑셀 파일을 바탕화면에 놓기
2. 터미널 켜기
3. `npx carter-picker` 실행
